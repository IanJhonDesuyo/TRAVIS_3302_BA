# TRAFFIQ — AI-Based Traffic Violation Analytics and Decision Support System

A standalone Bootstrap 5 + Vanilla JS admin dashboard. No build step required.

## Quick start
Unzip and open `index.html` in any modern browser. All dependencies (Bootstrap, Bootstrap Icons, Chart.js, Inter font) load from CDN — internet connection required.

## Structure
- `index.html` — Dashboard overview
- `monitoring.html` — Live camera feeds & officer presence
- `violations.html` — Ticket records
- `payments.html` — Treasury payments
- `alerts.html` — Real-time event stream
- `analytics.html` — ML predictions (Random Forest, K-Means)
- `reports.html` — Report export (PDF / Excel / CSV)
- `users.html` — User management
- `settings.html` — System configuration
- `css/style.css` — Custom theme
- `js/app.js` — Sidebar, clock, chart defaults

## Theme
- Primary Dark Blue `#1e3a8a`
- Secondary Green `#16a34a`
- Accent Amber `#f59e0b`
- Danger Red `#dc2626`


## Panel-aligned updates
- Treasury personnel records violations and payments by scanning the issued paper receipt/ticket or using manual input.
- Analytics focuses on season prediction and location-based violation hotspots.
- Computer vision monitoring includes vehicle counting, congestion detection, officer presence monitoring, and potential collision alerts.
- Public website page is included for advisories, activities, and public notices.
